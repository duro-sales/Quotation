      Calculation Engine Logic

      • Original Procurement: Purchase Base + Input GST. This is the exact amount you pay the distributor to buy the materials.
      • To Customer: Selling Base + Output GST. This is the hardware portion you bill the client.
      • DURO Profit: (Customer Price excl. tax) - (Purchase Base excl. tax) + Total Service Revenue.
      • Net Payable GST: Total Output GST (Hardware + Service) minus Hardware ITC.
